# Citation format

- PDF/Buku/Manuskrip: `(Penulis, Tahun, hlm.)` when bibliographic metadata available. In-case of only file: `Filename.pdf p.<halaman>`.
- Jurnal/Web: provide full URL and source name. Example: `Al-Ishlah (Journal) - https://example.edu/article123`.
- Community source: label as `(versi tradisi/komunitas)` and include original link if available.

When responding, always include at least 2 sources for factual claims when possible: prefer one academic (journal/book) + one primary manuscript page.
