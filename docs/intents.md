# Intents (high-level)

- `Q&A`: direct factual questions about Panglima Mayu (who, what, when, where, why)
- `Timeline`: requests for chronology of events
- `Tokoh`: queries about related figures and roles in Kesultanan Sumbawa
- `Artefak`: ask about manuscripts, artefacts, lokasi penyimpanan
- `Compare`: compare versions across sources (manuscript vs journal vs web)
- `Terminology`: define local terms, place names, and script/aksara
- `Pedagogy`: material tailored for students (learning objectives, MCQ)
- `Scholarly`: essay-level responses with historiography and debates (Mahasiswa mode)
